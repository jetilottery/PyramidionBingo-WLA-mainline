Pyramidion Bingo

Contemplez le pouvoir et le mystère de l’Égypte ancienne dans le jeu Pyramidion Bingo. Le pyramidion scintillant tire 70 numéros entre 1 et 100. Gagnez un lot lorsqu’un bloc de couleur est complété. Mélangez les numéros et les lots pour tenter votre chance!

Comment miser

Coût du billet

Affiche le coût actuel du billet.

Appuyez sur la flèche gauche (–) pour diminuer le coût du billet.

Appuyez sur la flèche droite (+) pour augmenter le coût du billet.

Appuyez sur ACHETER pour débiter le prix du billet de votre solde et commencer la partie.

Comment jouer

Le but du jeu Pyramidion Bingo est d’obtenir tous les numéros requis pour compléter un bloc de couleur gagnant.

Le tableau de jeu contient des numéros de 1 à 100. Pour jouer, tirez 70 numéros à inscrire au tableau. Marquez n’importe quel bloc de couleur gagnant pour gagner le lot correspondant.

Le jeu prend fin lorsque tous les numéros tirés sont indiqués sur le tableau. Le jeu affiche un message indiquant le résultat de la partie.

Tirer

Appuyez sur TIRER pour tirer des numéros.

Lots

Appuyez sur LOTS pour voir les lots associés aux blocs de couleur.

Continuer

Appuyez sur CONTINUER pour voir les numéros et continuer à tirer.

Mélanger

Appuyez sur MÉLANGER pour mélanger les numéros ou les lots.

Vitesse

Appuyez sur VITESSE pour changer la vitesse à laquelle les numéros sont tirés. Vous avez quatre options de vitesse.

Règles

Toute utilisation inappropriée et toute défectuosité annulent jeux et paiements.

Les gains sont affichés en argent.

Les choix effectués par le joueur pendant le jeu n’ont aucune incidence sur le résultat de la partie.
