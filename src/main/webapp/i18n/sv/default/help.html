Pyramidion Bingo

Upplev det antika Egyptens makt och magi i Pyramidion Bingo! En glittrande pyramidion drar 70 nummer mellan 1 och 100. När ett färgblock fullbordas vinner du ett pris. Pröva lyckan genom att blanda numren och priserna!

Så här satsar du

Lottkostnad

Visar den aktuella lottkostnaden.

Tryck på vänsterpilen (–) för att minska lottkostnaden.

Tryck på högerpilen (+) för att öka lottkostnaden.

Tryck på KÖP för att dra lottkostnaden från saldot och börja spela.

Så här spelar du

Målet med Pyramidion Bingo är att markera alla nummer som krävs för att bilda ett vinnande färgblock.

Spelplanen innehåller nummer från 1 till 100. Du spelar genom att dra 70 nummer som markeras på spelplanen. När du markerat ett vinnande färgblock helt och hållet vinner du motsvarande pris.

Spelet avslutas när alla dragna nummer har indikerats på spelplanen. Ett meddelande dyker upp och visar spelets utfall.

Dra nummer

Tryck på DRA NUMMER för att dra nummer.

Priser

Tryck på PRISER för att se färgblockens priser.

Fortsätt

Tryck på FORTSÄTT för att se numren och fortsätta med dragningen.

Blanda

Tryck på BLANDA för att blanda numren eller priserna.

Hastighet

Tryck på HASTIGHET för att ändra dragningarnas hastighet. Det finns 4 olika hastigheter.

Regler

Missbruk eller funktionsfel ogiltigförklarar alla utbetalningar och spel.

Vinster visas i vald valuta.

Val som gjorts av spelare under spelets gång påverkar inte spelets utfall.
